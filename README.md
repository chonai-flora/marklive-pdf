# marklive-pdf

作成・編集したMarkdown文書をPDF化することができます。  

`<br>` タグを挿入することでページを分割できます。

数式は
$ c = \pm\sqrt{a^2 + b^2} $
のように一行で表示するか、

$
S_n=\displaystyle\sum_{k=1}^{n} (x_k)^2
\Rightarrow
S_n=\sum_{k=1}^{n} (x_k)^2 
\Rightarrow 
{}^{i}_{j}T^{k}_{h}
$

のように複数行にまたがって記述してください。

また、Markdownファイルをテキストエリアにドラッグ&ドロップすることも可能です。

エディタ部分は https://github.com/uiwjs/react-md-editor を参考にしています。
