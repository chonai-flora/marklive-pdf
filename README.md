# marklive-pdf
Markdown文書をPDF形式に変換できます。

## 機能
- `<br>` タグを挿入してページを分割できます。
- 数式を一行または複数行で表示可能です。
- Markdownファイルをテキストエリアにドラッグ&ドロップすることで、簡単にPDFに変換できます。

### 数式の例
一行の数式  
$ c = \pm\sqrt{a^2 + b^2} $

複数行の数式  
$
S_n=\sum_{k=1}^{n} (x_k)^2 
\Rightarrow 
{}^{i}_{j}T^{k}_{h}
$
